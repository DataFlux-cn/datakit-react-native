# react-native-mobile

基于观测云 Android iOS React Native SDK

## Installation

```sh
npm install @cloudcare/react-native-mobile
```

## Usage

```js
import {
  FTMobileReactNative,
  FTReactNativeLog,
  FTReactNativeTrace,
  FTReactNativeRUM,
  FTMobileConfig,
  FTLogConfig,
  FTTraceConfig,
  FTRUMConfig,
  MonitorType,
  TraceType
} from '@cloudcare/react-native-mobile';

// ...

```
